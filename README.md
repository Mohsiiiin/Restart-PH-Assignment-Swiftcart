undefined কী?
যখন কোনো variable declare করা হয় কিন্তু value assign করা হয় না, তখন সেটা undefined হয়।

map() কী কাজে ব্যবহার হয়?

map() ব্যবহার করা হয় একটা array এর প্রতিটা element পরিবর্তন করে নতুন array বানানোর জন্য।

== আর === এর পার্থক্য ?

== শুধু value compare করে,
=== value + type দুইটাই compare করে 

async/await এর গুরুত্ব কী API fetch করার সময়?

async/await ব্যবহার করা হয় asynchronous code কে synchronous এর মতো readable করে লেখার জন্য।

API call সবসময় asynchronous হয়, কারণ server থেকে data আসতে সময় লাগে।

Scope কী?

Scope মানে, একটা variable কোথা থেকে access করা যাবে, আর কোথা থেকে যাবে না।
